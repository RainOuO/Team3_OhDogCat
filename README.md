## 「怎麼辦寵物出遊，行程好難排？」→ 那就找「Oh！DogCat 」

### 功能大綱

- 註冊、登入、第三方 LinePay 登入
- 忘記密碼、信箱驗證
- 會員系統
- 購物車、訂單成立通知信
- LinePay 金流串接
- Google Map 行程規劃
- 票券細節、瀏覽
- 360 度環景及街景、衛星地圖
- 票券搜尋、篩選、排序、收藏
- 聊天室
- 社群分享
- 所見即所得文章編輯
- 響應式設計

<br>
